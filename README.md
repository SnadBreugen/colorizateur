# colorizateur

Cable color tool for Audiotool projects. Detects signal chains, lets you assign colors per chain or per device type, applies them to all cables in the chain.

Built on the Audiotool Nexus SDK.

## Setup

```bash
npm install
npm run dev
```

Open http://127.0.0.1:5173/ and login with your Audiotool account.

## Status

v0.1 — cable colors only. Channel-strip and timeline-region coloring planned.
